1 Продвинутая работа со списками

Вчера ты создал свой первый список. Сегодня разберемся, как эффективно управлять списками и использовать их для работы.

Типы списков:

Тип Что это Когда использовать
Статический Фиксированный набор контактов. Ты добавляешь и удаляешь вручную Для конкретных кампаний: "UAE Fitness Owners - Март 2026"
Динамический Автоматически обновляется на основе фильтров (как сохраненный поиск) Для постоянного мониторинга: "Все CEO в UAE с высоким Intent"

Действия со списками:

Действие Как сделать Стоимость
Добавить в последовательность Выбрать контакты → Add to Sequence Бесплатно
Обогатить данные Выбрать контакты → Enrich Кредиты за новые данные
Экспортировать в CSV Выбрать контакты → Export 1 кредит / контакт
Переместить в другой список Выбрать → Add to list (выбрать другой) Бесплатно (уже сохранены)
Удалить из списка Выбрать → Remove from list Бесплатно
Массовое выделение Чекбокс сверху → Select all (до 50,000) Зависит от действия

Стратегия организации списков:

1
Поиск — находим людей по персоне + фильтрам
2
Сырой список — сохраняем в список "[Регион] [Отрасль] - RAW"
3
Проверка — просматриваем, удаляем нерелевантных
4
Рабочий список — перемещаем проверенных в "[Регион] [Отрасль] - READY"
5
Отправка — экспортируем в Smartlead для email-кампании
Совет: используй RAW / READY систему

Никогда не отправляй письма по "сырому" списку. Всегда проверяй контакты вручную перед переносом в рабочий список. Это экономит деньги и защищает репутацию отправителя.

2 Data Health Center: панель здоровья данных

Data Health Center — это панель мониторинга качества твоих контактных данных. Она показывает, что нужно обновить, обогатить или исправить.

Как попасть:

  1. В левой навигации нажми Data enrichment
  2. Вверху выбери вкладку Data health center

Что показывает панель:

💼
Job Changes
Контакты, сменившие работу. % актуальных vs устаревших vs неизвестных
💳
Credit Usage
Сколько кредитов потрачено и на что: email, телефоны, экспорт
📧
Missing Emails
Контакты без email. Можно попробовать обогатить

Что означают цвета в диаграммах:

Цвет Значение Что делать
Зеленый / Синий Данные актуальны, email есть Ничего, все хорошо
Желтый Нужно обновление (смена работы, данные устарели) Запустить обогащение
Серый Неизвестно (Apollo не может определить) Попробовать обогатить или проверить вручную

Кнопки на каждой карточке:

Кнопка Что делает
View contacts Открывает список контактов с этой проблемой (например, все без email)
Schedule Настраивает автоматическое обогащение по расписанию
View credit usage Показывает детальную разбивку расходов кредитов

3 Обогащение: недостающие email и телефоны

Обогащение (Enrichment) — это процесс, когда Apollo ищет недостающие данные о контактах: email, телефон, актуальную должность.

Способы обогащения:

Способ Как Когда использовать
Вручную Открой список → выбери контакты без email → нажми Enrich Для маленьких списков (до 50 человек)
Через Data Health Center Data Health Center → Missing Emails → View contacts → Enrich Для обогащения всех контактов без email сразу
По расписанию Data Health Center → Missing Emails → Schedule Для автоматического обогащения (раз в неделю/месяц)
CSV загрузка Data enrichment → CSV → загрузить файл Если есть список контактов из другого источника

Настройка автоматического обогащения:

  1. Перейди в Data enrichment → Data health center
  2. На карточке Missing Emails нажми Schedule
  3. Выбери тип: Contacts
  4. Выбери действие: Missing Emails
  5. Установи лимит кредитов (например, 50 в неделю) — это защита от перерасхода
  6. Выбери частоту: Weekly (еженедельно)
  7. Нажми Save
Лимит кредитов обязателен!

Всегда устанавливай лимит кредитов при автоматическом обогащении. Без лимита Apollo может потратить все кредиты за одну сессию. Рекомендуем: не более 50 кредитов за одно обогащение.

Что происходит при обогащении?

Apollo проверяет свою базу данных и пытается найти email контакта. Успешность зависит от популярности компании и должности. В среднем Apollo находит email для 60-70% контактов. Если email не найден — кредит не тратится.

4 Смена работы (Job Changes): как использовать

Когда человек меняет работу, это один из лучших моментов для продажи. Вчера мы включили Job Change Alerts. Сегодня научимся с ними работать.

Почему смена работы важна:

Причина Как это помогает продавать
Новая роль = новые задачи Человек ищет инструменты и партнеров для достижения целей на новой позиции
Новый бюджет Часто на новой позиции выделяется бюджет на маркетинг
Старые связи обнулились Предыдущее агентство осталось в старой компании
Хотят показать результат Первые 90 дней — время доказать себя, поэтому готовы инвестировать

Как найти контакты со сменой работы:

  1. Перейди в Data enrichment → Data health center
  2. На карточке Job Changes нажми View contacts
  3. Apollo откроет поиск с фильтром "контакты, сменившие работу"
  4. Просмотри список — увидишь старую и новую должность/компанию

Или через поиск:

  1. Перейди в People
  2. Нажми More Filters
  3. Найди фильтр Job Changes
  4. Выбери "Changed jobs"
  5. Добавь другие фильтры (персона, локация) для сужения результатов

Что делать с найденными контактами:

Действие Как
Обновить контакт Выбрать → Job change actions → Update existing contacts
Создать новый контакт Если человек перешел в компанию-цель: Create new contact
Добавить в отдельный список Создай список "Job Changes - [Месяц]" для отдельной кампании
Идея для outreach

Письмо типа "Поздравляю с новой ролью в [Компания]! На новом месте часто нужно быстро показать результат в маркетинге — мы помогаем с этим компаниям в [отрасль]." Такие письма имеют в 3-5 раз более высокий отклик.

5 Практика: аудит и обогащение тестового списка

Задание 1: Аудит тестового списка
  1. Перейди в Lists и открой свой тестовый список из Дня 2
  2. Посмотри, у скольких контактов есть email (зеленая иконка)
  3. Посмотри, у скольких email отсутствует
  4. Запиши результат: например, "15 из 20 с email, 5 без"
Задание 2: Data Health Center
  1. Перейди в Data enrichment → Data health center
  2. Посмотри 3 карточки: Job Changes, Credit Usage, Missing Emails
  3. Запиши текущие показатели (%, количество)
  4. Нажми View contacts на карточке Missing Emails
  5. Посмотри, какие контакты без email — есть ли среди них твои из тестового списка?
Задание 3: Поиск смен работы
  1. Перейди в People
  2. Примени персону UAE Business Owner
  3. Добавь фильтр Job Changes → Changed jobs
  4. Сколько результатов? Просмотри 5-10 карточек
  5. Обрати внимание на старую и новую позицию — какие переходы ты видишь?
Не трать кредиты на обогащение сегодня

Сегодня мы только изучаем и просматриваем. Не нажимай "Enrich" на реальных контактах — это потратит кредиты. Просто ознакомься с интерфейсом и данными.

Самопроверка:
  1. Чем статический список отличается от динамического?
  2. Что нужно обязательно установить при автоматическом обогащении?
  3. Тратится ли кредит, если Apollo не нашел email при обогащении?
  4. Почему первые 90 дней на новой работе — лучшее время для продажи?
  5. Зачем нужна система RAW / READY для списков?
Нажми, чтобы увидеть ответы
  1. Статический — контакты добавляются вручную. Динамический — обновляется автоматически на основе фильтров
  2. Лимит кредитов — без него Apollo может потратить все кредиты за одну сессию
  3. Нет — кредит тратится только при успешном обогащении
  4. Новый бюджет, нужно показать результат, старые партнеры обнулились, открыты к новым предложениям
  5. Чтобы не отправлять письма непроверенным контактам — сначала проверяем (RAW), потом отправляем (READY)
